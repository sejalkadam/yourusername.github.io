VeritasIQ — Verified Intelligence

GenAI · RAG · Semantic Search · Vector DB · NLP

verified

A RAG-based verification engine that cross-validates company data across 5 independent sources — exchange filings, Wikipedia, news, ratios — and tags every claim Verified / Conflicting / Unverified, instead of trusting a single source or raw LLM output. A semantic claim-matching and scoring pipeline (LangChain, ChromaDB, Sentence-Transformer embeddings, cosine similarity, 8% tolerance) resolves cross-source conflicts into one verification score, which a Groq LLM then uses to ground SWOT / Porter's / Bull-vs-Bear narratives — cutting hallucination risk at the source.
